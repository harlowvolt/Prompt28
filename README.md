# Orion Orb (Prompt28)

Orion Orb is an iOS app that turns rough ideas, voice notes, and messy drafts into stronger AI-ready prompts.

Built for fast prompt transformation, the app combines guest-friendly onboarding, prompt refinement tools, reusable templates, and native sharing in a polished SwiftUI experience.

## Key Features

- AI prompt transformation from freeform text or voice input
- One-tap prompt refinements like more professional, more detailed, shorter, and more persuasive
- Templates and use cases across Business, School, Creative, Marketing, and Personal categories
- Native share cards and ShareLink support for smooth iPhone and iPad sharing
- Guest mode, auth flow, and account deletion backed by Supabase
