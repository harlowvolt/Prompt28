# Orion Orb (Prompt28)

Orion Orb (Prompt28) is an AI prompt transformation tool for iPhone and iPad that turns rough ideas, voice notes, and messy drafts into stronger prompts for modern AI workflows.

Built for fast prompt transformation, the app combines an AI consent flow, guest-friendly onboarding, one-tap refinements, reusable templates, native sharing, and a Supabase-backed account system in a polished SwiftUI experience.

## Key Features

- AI prompt transformation from freeform text or voice input
- Required AI consent flow before entering the main app
- One-tap prompt refinements like more professional, more detailed, shorter, and more persuasive
- Templates and use cases across Business, School, Creative, Marketing, and Personal categories
- Native share cards and ShareLink support for smooth iPhone and iPad sharing
- Guest mode, authentication, account deletion, and backend sync powered by Supabase
